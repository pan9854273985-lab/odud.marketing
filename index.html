Что нужно

Требования

Минимум технического. Всё остальное объясняем в первом модуле.

Подписка Claude Pro или Max
От $20/мес. Зарегистрироваться на claude.ai. Бесплатный тариф не поддерживается.
Node.js 18+ (опционально)
Нужен только при установке через терминал. Десктопное приложение Claude — без него.
macOS или Windows 10+
Инструкции есть для обеих платформ. Linux тоже подходит.
Опыт в маркетинге
Базовое понимание маркетинговых задач. Программирование не нужно — совсем.
