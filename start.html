Установка и запуск курса

Выбери среду и операционную систему — покажем точные шаги. Курс одинаковый, отличается только инструмент.

Среда
Система

Шаг 1 ⏱ ~2 минуты

Установи приложение Claude

Скачай десктопное приложение с claude.ai/download и установи его.

Нужна подписка Claude Pro или Max (от $20/мес) — бесплатный тариф не поддерживается.

Шаг 2 ⏱ ~2 минуты

Распакуй файлы курса

Скачай pulse-course.zip из закреплённого сообщения в Telegram-канале.

Дважды кликни по архиву — он распакуется автоматически. Перемести папку pulse-course в «Документы».

Шаг 3 ⏱ ~1 минута

Открой курс

  1. Открой приложение Claude → перейди на вкладку Code
  2. Нажми + и выбери папку pulse-course
  3. В поле ввода напиши /go-welcome и нажми Enter

Курс начнётся автоматически.

Готово — начинаем курс

Claude проведёт тебя через онбординг — расскажет как устроен курс, как возвращаться и убедится что всё готово. Займёт 5 минут.

Шаг 1 ⏱ ~15 минут

Установка Claude Code

Что нужно

ТребованиеПодробности
Подписка Claude Pro или Max От $20/мес. Зарегистрироваться: claude.ai. Бесплатный тариф не поддерживается.
Node.js 18+ Бесплатно. Скачать: nodejs.org
macOS Любая версия macOS 11+

Как открыть Терминал

  1. Нажми Cmd + Space — откроется Spotlight
  2. Введи «Терминал» и нажми Enter

Шаг 1. Проверь Node.js

node --version

Если видишь v18.0.0 или выше — переходи к шагу 3. Если команда не найдена — переходи к шагу 2.

Шаг 2. Установи Node.js (если нужно)

Скачай LTS-версию с nodejs.org и запусти установщик. Или через Homebrew:

brew install node
Важно: после установки закрой Терминал и открой новый.

Шаг 3. Установи Claude Code

curl -fsSL https://claude.ai/install.sh | bash

Дождись сообщения Installation complete!.

🇷🇺 Если видишь HTML-код вместо установки — это геоблокировка. Используй npm: 
npm install -g @anthropic-ai/claude-code
Для первого входа в аккаунт может потребоваться VPN — только на время авторизации в браузере.

Шаг 4. Проверь установку

claude --version

Если видишь номер версии — готово. Если command not found — закрой Терминал, открой новый.

Шаг 5. Войди в аккаунт

claude
  1. Выбери «Учётная запись Claude с подпиской» (не Anthropic Console)
  2. Войди через браузер
  3. Выбери тему — Dark mode или любую другую, нажми Enter

Шаг 6. Проверь что всё работает

/doctor

Всё должно быть отмечено зелёными галочками.

Устранение проблем

ОшибкаЧто делать
command not found: claude Закрой Терминал, открой новый. Если не помогает — добавь в ~/.zshrc: export PATH=$HOME/.local/bin:$PATH, потом source ~/.zshrc
«Требуется подписка» Убедись что подписка активна на claude.ai.
«Новый терминал в папке» не появляется в Finder Finder → Настройки → Службы → включи «Новый терминал в папке»
🥚 Ничего не помогает Открой claude.ai и спроси прямо там: «Я пытаюсь запустить Claude Code на Mac, вот что происходит: [опиши ошибку]. Что не так и как исправить?» — Claude объяснит.
